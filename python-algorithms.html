Python Algorithms for Automation

In this lab, the goal was to create an algorithm that would:
1. Check if a listed IP address from a text file called allow_list was also in a list of IP addresses to remove called remove_list.
2. If the IP address was in remove_list, the algorithm would use the .remove() method to remove it from the allowed list.
After running the algorithm and removing all the excluded IPs, then I used the .join() method to modify the list into string data. After that step, I overwrote the original list of allowed IP addresses so that it was current.

Open the file that contains the allow list

The first line of code is assigning a variable called import_file to the name of our allow list text file.
The second line is assigning a variable remove_list to a list of IP addresses to remove.
The final line is the with statement to open the allow list file.

Read the file contents

This step completes the with statement by reading the contents of the file. I have saved the contents in a variable called ip_addresses and used the print function.

Convert the string into a list

I used the .split() method to separate the file into a list in Python. This allows me to work with the data in separated elements, and to create the algorithm in later steps.

Iterate through the remove list

I created a for loop with a loop variable called element which interates through the ip_addresses list. Then I have used the print function to show each element.

Remove IP addresses that are on the remove list

After starting the for loop, I created an if statement to check if each element is in the remove_list. If so, then Python executes the .remove() method. Finally I used the print function to display the list after removing all excluded IP addresses. The list went from 17 addresses to 13 after removing them.

Update the file with the revised list of IP addresses

Now I need to convert the list back into string data so that I can overwrite the original text file. I used the .join() method for this. Then I used another with statement to write in the file. When this is executed it overwrites the previous data with our updated import_file containing the allowed list inside ip_addresses. The last part of the code would allow me to read and display the file contents using print(text)

Summary

I used with statements to open, read, and write a text file. The print function was used to display the contents before and after editing. The .split() method was used to create a list that can be modified inside Python. I created an algorithm using a for loop and checking if each element was in remove_list, and if it was, I instructed to use the .remove() method. Finally I needed to overwrite the file with the updated data, which first required using .join() to convert the list back into string data. I used .write() to overwrite the file, and .read() so it could be displayed with the new data.
